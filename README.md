# Détection de Toxicité dans les Commentaires

Ce projet implémente un système de détection de toxicité dans les commentaires en utilisant des algorithmes de Machine Learning (Naive Bayes, Régression Logistique) implémentés de zéro et comparés avec Scikit-Learn.

## Structure du Projet

- `src/` : Contient le code source (prétraitement, vectorisation, modèles, visualisation).
- `sklearn_models.py` : Implémentation utilisant Scikit-Learn.
- `main.py` : Point d'entrée principal.
- `requirements.txt` : Liste des dépendances.

## Algorithmes
- Naive Bayes (From scratch & Scikit-Learn)
- Logistic Regression (From scratch & Scikit-Learn)

## Installation
```bash
pip install -r requirements.txt
```
